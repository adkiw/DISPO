# DISPO – Krovinių valdymo sistema

Ši Streamlit programa leidžia:
- Įrašyti krovinio duomenis (pakrovimas, iškrovimas, km, frachtas)
- Peržiūrėti įrašytų krovinių lentelę

## Paleidimas internete

1. Įkelk šį projektą į GitHub
2. Eik į https://streamlit.io/cloud ir prisijunk
3. Pasirink "Deploy from GitHub", įvesk repo adresą
4. Nurodyk failą: `main.py`

Ir tavo programa veiks web'e!
